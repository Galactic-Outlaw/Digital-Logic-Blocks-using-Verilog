File names and their description

ff.v : Full Adder
faha.v : Full Adder using Half Adder
bshifter_4bit.v : 4 bit barrel Shifer 
array_mul.v : Array Multiplier 
add_sub_4bit.v : 4 bit Adder Subtractor
